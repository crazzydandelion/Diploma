# Дипломная работа по курсу "Инженер по тестированию"
## Автоматизация UI и API тестов для сайта Кинопоиска
### Описание базового синтаксиса записи и форматирования

Проект использует следующие технологии и синтаксис:

1. **Python**: Основной язык программирования для написания тестов.
2. **Selenium**: Библиотека для автоматизации взаимодействия с веб-браузером.
3. **Pytest**: Фреймворк для написания и запуска тестов.
4. **Allure**: Инструмент для генерации отчетов о выполнении тестов.

### Форматирование кода

- Код форматируется в соответствии с PEP 8 (стиль написания кода на Python).
- Используются docstrings для документирования методов и функций.
- Все шаги теста размечаются с помощью `@allure.step` или `with allure.step` для улучшения читаемости отчетов.

## Структура проекта
### Проект состоит из:
- двух тестовых файлов с тестами UI и API соответственно;
- файлов с описанием страниц, используемых в тестах, в соответствии с паттерном PageObject;
- файла с зависимостями requirements.txt;
- файла README с описанием проекта.

Представленные в проекте тесты являются
автоматизированной версией функциональных тестов по фронтенду Кинопоиска (UI)
и API-тестов его сервера, которые были проведены в Финальной работе 
по ручному тестированию 
(https://my.sky.pro/student-cabinet/stream-module/23364/course-final-work/submit-work)

### UI:
- Поиск фильма с параметризацией;
- Открытие вкладки "Билеты в кино";
- Открытие вкладки "Вакансии";
- Открытие диалогового окна со службой поддержки;
- Проход по ссылкам: "Медиа" -> "Рубрики" -> "Привет, меня зовут..."

### API:

#### Позитивные тесты:
- Поиск фильма по ID;
- Поиск рандомного тайтла из базы;
- Поиск по фильтрам;

#### Негативные тесты:
- Поиск заведомо невалидного ID;
- Поиск с неправильным методом (POST);
- Поиск без токена.

## Инструкция по запуску тестов для формирования отчета Allure

1. Установите необходимые зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Запустите тесты:
   ```bash
   pytest --alluredir=./allure-results
   ```
   Эта команда запустит все тесты и сохранит результаты в директорию `./allure-results`.
---
## Инструкция по просмотру сформированного отчета Allure

1. После завершения тестов, чтобы просмотреть отчет, выполните команду:
   ```bash
   allure serve allure-results
   ```
   Эта команда запустит локальный сервер и откроет отчет в браузере.
